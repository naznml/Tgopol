1.1 Натурал сандар

Натурал сандар жиыны шексіз жалғасады: 1, 2, 3, 4, 5, 6, ... . Әрбір келесі натурал сан алдыңғы саннан 1-ге артық болады.

Натурал сандар қатары
1, 2, 3, 4, 5, 6, ...

Санның разрядтық құрамы оның қандай жүздіктерден, ондықтардан және бірліктерден құралғанын көрсетеді.

1.1 Натуральные числа

Натуральные числа используются для счёта. Когда считают предметы, используют числа 1, 2, 3, 4, 5. Такие числа называются натуральными.

Ряд натуральных чисел продолжается бесконечно: 1, 2, 3, 4, 5, 6, ... . Каждое следующее натуральное число на 1 больше предыдущего.

Ряд натуральных чисел
1, 2, 3, 4, 5, 6, ...

Разрядный состав числа показывает, из каких сотен, десятков и единиц состоит число.

Разложение по разрядам
347 = 300 + 40 + 7

В числе 347 есть 3 сотни, 4 десятка и 7 единиц. 300 — разряд сотен, 40 — разряд десятков, 7 — разряд единиц.

Пример 1

Разложите число 586 на разрядные слагаемые.

В числе 586 есть 5 сотен, значит часть сотен — 500.

В числе 586 есть 8 десятков, значит часть десятков — 80.

В числе 586 есть 6 единиц, значит часть единиц — 6.

Ответ: 586 = 500 + 80 + 6.

Пример 2

Запишите числа 42, 17, 95, 8 в порядке возрастания.

Порядок возрастания начинается с самого маленького числа.

Самое маленькое число из данных — 8.

После него идёт 17, затем 42, самое большое — 95.

Ответ: 8, 17, 42, 95.

Пример 3

A = 6 × 8, B = 50. Сравните A и B.

Сначала находим значение A: 6 × 8 = 48.

Теперь сравниваем 48 и 50.

Число 48 меньше числа 50.

Ответ: A < B.

Пример 4

Определите цифру в разряде десятков в числе 729.

Разложим число 729 по разрядам: 700 + 20 + 9.

700 — сотни, 20 — десятки, 9 — единицы.

Число 20 состоит из 2 десятков.

Ответ: цифра в разряде десятков — 2.

Натурал сандар бойынша тест

1. 908 санын разрядтық қосылғыштарға жіктеңіз.

908 санында 9 жүздік бар, сондықтан жүздік бөлігі — 900. Ондық разрядтағы цифр 0, сондықтан 10 немесе 90 қосылмайды. Бірлік разрядтағы цифр 8, сондықтан бірлік бөлігі — 8. Демек, 908 = 900 + 8.

2. 16, 61, 6, 106 сандарын өсу ретімен орналастырыңыз.

Өсу реті ең кіші саннан басталады. Берілген сандар: 16, 61, 6, 106. Ең кішісі — 6. Одан кейін 16, содан кейін 61, ең үлкені 106. Сондықтан дұрыс рет: 6, 16, 61, 106.

3. A = 7 × 9, B = 64. Дұрыс салыстыруды таңдаңыз.

Алдымен A мәнін табамыз: 7 × 9 = 63. B мәні 64-ке тең. Енді 63 пен 64-ті салыстырамыз. 63 саны 64-тен кіші. Сондықтан A < B.

4. 438 санындағы жүздік разрядтағы цифрды табыңыз.

438 санын разрядтарға бөлеміз: 400 + 30 + 8. 400 — жүздік бөлік. Бұл бөліктегі цифр 4. Сондықтан жүздік разрядтағы цифр — 4.

5. Қай сан натурал санға жатады?

Натурал сандар санау үшін қолданылады: 1, 2, 3, 4, ... . Теріс сандар натурал сан емес. Бөлшек және ондық бөлшек те натурал сан емес. 12 саны санауда қолданылатын бүтін оң сан, сондықтан ол натурал сан.

Тест по натуральным числам

1. Разложите число 908 на разрядные слагаемые.

В числе 908 есть 9 сотен, поэтому часть сотен — 900. В разряде десятков стоит 0, поэтому 10 или 90 не добавляются. В разряде единиц стоит 8, поэтому часть единиц — 8. Значит, 908 = 900 + 8.

2. Расположите числа 16, 61, 6, 106 в порядке возрастания.

Порядок возрастания начинается с самого маленького числа. Даны числа 16, 61, 6, 106. Самое маленькое — 6. Затем идёт 16, потом 61, самое большое — 106. Правильный порядок: 6, 16, 61, 106.

3. A = 7 × 9, B = 64. Выберите верное сравнение.

Сначала найдём значение A: 7 × 9 = 63. Значение B равно 64. Сравниваем 63 и 64. Число 63 меньше числа 64. Значит, A < B.

4. Найдите цифру в разряде сотен в числе 438.

Разложим число 438 по разрядам: 400 + 30 + 8. Число 400 относится к сотням. Цифра в разряде сотен — 4.

5. Какое число является натуральным?

Натуральные числа используются для счёта: 1, 2, 3, 4, ... . Отрицательные числа не являются натуральными. Дроби и десятичные дроби тоже не являются натуральными. Число 12 — положительное целое число, которое можно использовать при счёте, поэтому оно натуральное.

1.2 Бүтін сандар

Бүтін сандарға оң сандар, теріс сандар және нөл жатады. Оң сандар нөлден үлкен болады, ал теріс сандар нөлден кіші болады.

Бүтін сандар сан түзуінде орналасады. Сан түзуінде оңға қарай жүрген сайын сандар үлкейеді, солға қарай жүрген сайын сандар кішірейеді.

Бүтін сандар қатары
..., -4, -3, -2, -1, 0, 1, 2, 3, 4, ...

Қарама-қарсы сандардың таңбалары әртүрлі, бірақ нөлден қашықтығы бірдей болады. Мысалы, 5 және -5 — қарама-қарсы сандар.

Қарама-қарсы сандар
a және -a

Мысалы, 7 санының қарама-қарсы саны -7, ал -12 санының қарама-қарсы саны 12 болады.

Мысал 1

-3 және 2 сандарын салыстырыңыз.

-3 саны теріс сан, ал 2 саны оң сан.

Кез келген теріс сан кез келген оң саннан кіші болады.

Сондықтан -3 саны 2 санынан кіші.

Жауап: -3 < 2.

Мысал 2

-8, -2, 0, 5 сандарын өсу ретімен жазыңыз.

Өсу реті ең кіші саннан басталады.

Теріс сандардың ішінде нөлден алысырақ орналасқан сан кішірек болады.

-8 саны -2 санынан кіші, себебі -8 сан түзуінде сол жақта орналасады.

Одан кейін 0, содан кейін оң сан 5 келеді.

Жауап: -8, -2, 0, 5.

Мысал 3

-15 санының қарама-қарсы санын табыңыз.

Қарама-қарсы санды табу үшін санның таңбасын өзгертеміз.

-15 саны теріс таңбамен берілген.

Оның таңбасын оң таңбаға өзгертеміз.

Жауап: 15.

Мысал 4

A = -4 + 9, B = 3. A мен B шамаларын салыстырыңыз.

Алдымен A мәнін табамыз.

-4 + 9 дегеніміз: 9 санынан 4-ті азайтамыз, себебі таңбалары әртүрлі.

9 - 4 = 5, үлкен санның таңбасы оң болғандықтан нәтиже 5 болады.

Енді A = 5, B = 3.

5 саны 3-тен үлкен.

Жауап: A > B.

1.2 Целые числа

К целым числам относятся положительные числа, отрицательные числа и ноль. Положительные числа больше нуля, а отрицательные числа меньше нуля.

Целые числа располагаются на числовой прямой. Чем правее число находится на числовой прямой, тем оно больше. Чем левее число находится, тем оно меньше.

Ряд целых чисел
..., -4, -3, -2, -1, 0, 1, 2, 3, 4, ...

Противоположные числа имеют разные знаки, но находятся на одинаковом расстоянии от нуля. Например, 5 и -5 — противоположные числа.

Противоположные числа
a и -a

Например, противоположное число для 7 — это -7, а противоположное число для -12 — это 12.

Пример 1

Сравните числа -3 и 2.

-3 — отрицательное число, а 2 — положительное число.

Любое отрицательное число меньше любого положительного числа.

Значит, -3 меньше 2.

Ответ: -3 < 2.

Пример 2

Запишите числа -8, -2, 0, 5 в порядке возрастания.

Порядок возрастания начинается с самого маленького числа.

Среди отрицательных чисел меньше то число, которое дальше от нуля влево.

-8 меньше -2, потому что -8 находится левее на числовой прямой.

После них идёт 0, затем положительное число 5.

Ответ: -8, -2, 0, 5.

Пример 3

Найдите противоположное число для -15.

Чтобы найти противоположное число, нужно изменить знак числа.

Число -15 дано с отрицательным знаком.

Меняем знак на положительный.

Ответ: 15.

Пример 4

A = -4 + 9, B = 3. Сравните A и B.

Сначала найдём значение A.

-4 + 9: знаки разные, поэтому из большего по модулю числа 9 вычитаем 4.

9 - 4 = 5. У большего числа знак положительный, поэтому результат равен 5.

Теперь A = 5, B = 3.

5 больше 3.

Ответ: A > B.

Бүтін сандар бойынша тест

1. Қай қатарда бүтін сандар ғана берілген?

Бүтін сандарға бөлшексіз сандар жатады: теріс бүтін сандар, нөл және оң бүтін сандар. 1/2, 0,8, 2/3, 7,5 сияқты сандар бүтін емес. -9, -1, 0, 6 сандарының бәрі бөлшексіз жазылған, сондықтан бұл қатарда бүтін сандар ғана берілген.

2. -6 және -2 сандарын салыстырыңыз.

Екі сан да теріс. Теріс сандарды салыстырғанда нөлден алысырақ жатқан сан кіші болады. -6 саны сан түзуінде -2 санынан сол жақта орналасады. Сол жақтағы сан кіші болғандықтан, -6 < -2.

3. -11 санының қарама-қарсы санын табыңыз.

Қарама-қарсы санды табу үшін санның таңбасын өзгертеміз. -11 саны теріс таңбамен берілген. Оның таңбасын оңға ауыстырсақ, 11 шығады. Сондықтан -11 санының қарама-қарсы саны — 11.

4. -4, 3, -1, 0 сандарын өсу ретімен орналастырыңыз.

Өсу реті ең кіші саннан басталады. Теріс сандардың ішінде -4 саны -1 санынан кіші. Одан кейін 0 келеді, себебі 0 теріс сандардан үлкен. Соңында 3 келеді, себебі ол оң сан. Дұрыс рет: -4, -1, 0, 3.

5. A = -7 + 12, B = 4. Дұрыс салыстыруды таңдаңыз.

Алдымен A мәнін табамыз: -7 + 12. Таңбалары әртүрлі болғандықтан, 12-ден 7-ні азайтамыз: 12 - 7 = 5. Үлкен сан 12 оң таңбалы болғандықтан, нәтиже 5 болады. Енді A = 5, B = 4. 5 саны 4-тен үлкен, сондықтан A > B.

6. -8 + 3 өрнегінің мәнін табыңыз.

-8 + 3 өрнегінде сандардың таңбалары әртүрлі. Сондықтан үлкен модульден кіші модульді азайтамыз: 8 - 3 = 5. Үлкен модуль 8 санына тиесілі, оның таңбасы теріс. Сондықтан нәтиже -5 болады.

7. -5 - 4 өрнегінің мәнін табыңыз.

-5 - 4 дегеніміз -5 санына тағы 4 бірлікке азаю дегенді білдіреді. Сан түзуінде -5-тен солға қарай 4 қадам жүреміз: -6, -7, -8, -9. Сондықтан нәтиже -9 болады.

8. Қай тұжырым дұрыс?

Оң бүтін сандар 1, 2, 3, 4 сияқты сандар. Олардың бәрі нөлден оң жақта орналасады, сондықтан нөлден үлкен болады. Теріс сандар нөлден кіші. Сондықтан дұрыс тұжырым: кез келген оң бүтін сан нөлден үлкен.

9. 6 - 14 өрнегінің мәнін табыңыз.

6 - 14 дегеніміз 6 санынан 14-ті азайту. 14 саны 6-дан үлкен, сондықтан нәтиже теріс болады. Айырманың шамасын табамыз: 14 - 6 = 8. Сондықтан 6 - 14 = -8.

10. A = -2 · 6, B = -15. Дұрыс салыстыруды таңдаңыз.

Алдымен A мәнін табамыз: -2 · 6 = -12. Енді A = -12, B = -15. Теріс сандарды салыстырғанда нөлге жақынырақ сан үлкен болады. -12 саны -15-тен нөлге жақын. Сондықтан -12 > -15, яғни A > B.

Тест по целым числам

1. В каком ряду даны только целые числа?

Целые числа — это числа без дробной части: отрицательные целые числа, ноль и положительные целые числа. Числа 1/2, 0,8, 2/3, 7,5 не являются целыми. В ряду -9, -1, 0, 6 все числа записаны без дробной части, поэтому это целые числа.

2. Сравните числа -6 и -2.

Оба числа отрицательные. При сравнении отрицательных чисел меньше то число, которое находится дальше от нуля влево. Число -6 находится левее числа -2 на числовой прямой. Поэтому -6 < -2.

3. Найдите противоположное число для -11.

Чтобы найти противоположное число, нужно изменить знак числа. Число -11 дано с отрицательным знаком. Если поменять знак на положительный, получится 11. Значит, противоположное число для -11 — это 11.

4. Расположите числа -4, 3, -1, 0 в порядке возрастания.

Порядок возрастания начинается с самого маленького числа. Среди отрицательных чисел -4 меньше, чем -1. После них идёт 0, потому что 0 больше отрицательных чисел. Затем идёт положительное число 3. Правильный порядок: -4, -1, 0, 3.

5. A = -7 + 12, B = 4. Выберите верное сравнение.

Сначала найдём A: -7 + 12. Знаки разные, поэтому из 12 вычитаем 7: 12 - 7 = 5. Большее число 12 имеет положительный знак, поэтому результат равен 5. Теперь A = 5, B = 4. Число 5 больше числа 4, значит A > B.

6. Найдите значение выражения -8 + 3.

В выражении -8 + 3 числа имеют разные знаки. Поэтому из большего модуля вычитаем меньший: 8 - 3 = 5. Больший модуль принадлежит числу -8, у него отрицательный знак. Поэтому результат равен -5.

7. Найдите значение выражения -5 - 4.

-5 - 4 означает, что от числа -5 нужно уменьшиться ещё на 4 единицы. На числовой прямой от -5 идём на 4 шага влево: -6, -7, -8, -9. Поэтому результат равен -9.

8. Какое утверждение верно?

Положительные целые числа — это 1, 2, 3, 4 и так далее. Все они находятся правее нуля на числовой прямой, поэтому они больше нуля. Отрицательные числа меньше нуля. Верное утверждение: любое положительное целое число больше нуля.

9. Найдите значение выражения 6 - 14.

6 - 14 означает, что из 6 нужно вычесть 14. Число 14 больше числа 6, поэтому результат будет отрицательным. Находим разницу: 14 - 6 = 8. Значит, 6 - 14 = -8.

10. A = -2 · 6, B = -15. Выберите верное сравнение.

Сначала найдём A: -2 · 6 = -12. Теперь A = -12, B = -15. При сравнении отрицательных чисел больше то число, которое ближе к нулю. Число -12 ближе к нулю, чем -15. Значит, -12 > -15, поэтому A > B.
